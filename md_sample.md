# はじめに
ブログ記事を書く際の記法として、マークダウンとMBSの2つの選択肢があります。この記事では、それぞれの特徴と使い方について解説していきます。

## 基本的な使い方
通常のテキストはこのように書きます。**強調したいテキスト**は前後にアスタリスクを2つ付けます。

### コードブロックの使用例
コードブロックは以下のように記述します：

```python
def hello():
    print("Hello, World!")
    return True
```
